1. Открыть 2 cmd
    (действия со 2 по 4 выполнить в обоих cmd одновременно)
    в ковычках написаны команды, ковычки в cmd писать не нужно

2. выполнить команду "cd (путь до основной папки pithin)"
    например у меня cd c:/Users/fedor/Desktop/pithin

3. выполнить команду "venv/Scripts/activate.bat"
    после чего слева появить надпись: (venv)

4. выполнить команду "cd src"

5. в одной консоле выполнить команду "python server/server.py" (запуск сервера),
    а в другой "python client/client.py" (запуск клиента)
    у cmd клиента появится: >> 

!!!Важно!!!
команды вводить с маленькой буквы
test.csv документ с данными
server.py скрипт сервера обрабатывающего команды клиента
client.py скрипт клиента для отправления запросов серверу
data_handler.py специальный скрипт написанный для работы с csv документами, а также для красивого вывода таблиц
папка venv виртуальное окружение с нужной версией питона и библиотеками
